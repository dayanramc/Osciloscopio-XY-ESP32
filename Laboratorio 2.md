# Laboratorio 2 - Osciloscopio XY con ESP32

## Introducción

En este laboratorio se desarrolló un sistema capaz de dibujar figuras en un osciloscopio en modo XY utilizando un microcontrolador ESP32, convertidores digital-analógico (DAC) y un circuito de acondicionamiento con amplificadores operacionales.

---

## Objetivos

- Generar señales analógicas en los ejes X y Y  
- Implementar control mediante potenciómetros  
- Visualizar figuras en un osciloscopio  
- Comprender el uso de DAC y multiplexores  

---

## Materiales

- ESP32  
- 2x MCP4725 (DAC)  
- CD74HC4067 (Multiplexor)  
- LM324N (Amplificador operacional)  
- 3 Potenciómetros  
- Protoboard  
- Cables  

---

## Conexión del sistema

### ESP32 → DAC

| ESP32 | DAC |
|------|-----|
| GPIO21 | SDA |
| GPIO22 | SCL |
| 3V3 | VCC |
| GND | GND |

---

### ESP32 → Multiplexor

| ESP32 | MUX |
|------|-----|
| GPIO18 | S0 |
| GPIO19 | S1 |
| GPIO23 | S2 |
| GPIO5  | S3 |
| GPIO34 | SIG |

---

### Potenciómetros

- POT1 → C0 (Escala X)  
- POT2 → C1 (Escala Y)  
- POT3 → C2 (Velocidad)  

---

## Funcionamiento

El sistema genera coordenadas (X,Y) mediante el ESP32.  
Estas coordenadas se convierten a señales analógicas con los DAC y se envían al osciloscopio.

Los potenciómetros permiten modificar:

- Tamaño del dibujo  
- Velocidad de trazado

// Configuración de velocidad (en microsegundos)
const int tiempoPunto = 200; 

void setup() {
  // En el ESP32, dacWrite no requiere configurar pinMode
}

void loop() {
  // Recorremos cada punto de la trayectoria
  for (int i = 0; i < numPuntos; i++) {
    // Enviamos el valor X al pin 25
    dacWrite(DAC_X, trayectoria[i].x);
    // Enviamos el valor Y al pin 26
    dacWrite(DAC_Y, trayectoria[i].y);
    
    // Esperamos un poco para que el haz del osciloscopio se mueva
    delayMicroseconds(20);
  }
}
